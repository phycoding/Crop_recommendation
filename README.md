### Installation

How to install the required libraries for completing the setup.

1. Clone the repo
   ```sh
   git clone https://github.com/phycoding/Crop_recommendation.git
   ```
2. Install streamlit
   ```sh
   pip install streamlit
   ```
3. Run the app file
   ```js
   streamlit run Crop_app.py;
   ```
